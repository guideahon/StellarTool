Main file - Stellar Tool

The full tool (GUI + headless CLI), self-contained: Qt runtime plus the open-source community CLI binaries it drives, unmodified — repak and retoc (trumank), UAssetGUI (atenfyr) and cue4parse (CUE4Parse CLI, reads Zen mods). Extract anywhere and run StellarTool.exe. No installer, nothing written to the game folder; the merged output goes wherever you choose. Full source and build instructions: github.com/guideahon/StellarTool — you can compile it yourself instead.
